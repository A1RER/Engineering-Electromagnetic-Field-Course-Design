# 电偶极子与磁偶极子空间场强计算及可视化讲解稿

> 建议时长：3 到 5 分钟  
> 适用内容：Matlab / C 语言仿真程序展示  
> 讲解重点：计算模型、程序流程、可视化结果、与课程要求的对应关系

## 1. 开场介绍

大家好，我这次课程设计的题目是：利用 Matlab 或 C 语言计算电偶极子的空间电场强度，并画出电场分布图；同时计算磁偶极子的空间磁场强度，并画出磁场分布图。

我的实现主要分为两个部分：一部分是 Matlab 程序，用来完成场强计算和图形可视化；另一部分是 C 语言程序，用来完成三维空间网格上的数值计算，并把结果导出为 CSV 文件，方便后续用 Origin 等软件绘图。

整体来说，程序不仅完成了题目要求中的“计算”和“画图”，还额外加入了数据导出、解析公式验证和三维切片显示等功能。这些扩展内容并没有改变题目方向，而是为了更清楚地展示偶极子场的空间分布特征。

## 2. 物理模型说明

首先介绍一下使用的物理模型。

对于电偶极子，我把它看作一个位于坐标原点、方向沿 z 轴的偶极矩。程序中使用的电偶极矩为：

```text
p = [0, 0, pz]
```

静态电偶极子的电场可以写成：

```text
E = 1 / (4*pi*epsilon_0) * [3*r*(p·r)/r^5 - p/r^3]
```

这里 `epsilon_0` 是真空介电常数，`r` 是从偶极子源点指向观察点的位置矢量。

磁偶极子的处理方法与电偶极子类似，只是把电偶极矩换成磁偶极矩，计算的物理量从电场强度 E 变为磁场强度 H。静态磁偶极子的空间分布形式和电偶极子在数学结构上非常相似，因此可以使用同一套网格和可视化流程。

由于偶极子在源点附近存在理论奇异性，也就是距离趋近于零时场强会趋于无穷大，所以程序中加入了一个很小的软化因子，避免数值计算时出现除零或无穷大的问题。这是数值仿真中比较常见的处理方式。

## 3. 程序计算流程

接下来说明程序的主要流程。

第一步是设置物理常数和仿真参数，包括真空介电常数、真空磁导率、偶极矩方向、计算区域大小和网格密度等。

第二步是在空间中建立计算网格。Matlab 程序主要使用 xz 平面作为二维切片来展示场分布，也可以生成三维网格用于切片显示。C 语言程序则通过三重循环遍历三维网格中的每一个点，逐点计算场强分量。

第三步是计算每个空间点上的场强。程序输出的不只是场强大小，还包括三个方向分量：

```text
Fx, Fy, Fz, F_mag
```

其中对于电偶极子，`F` 表示电场 E；对于磁偶极子，`F` 表示磁场 H。`F_mag` 表示场强模值，也就是空间某一点场强的大小。

第四步是进行后处理和画图。Matlab 程序会绘制场强幅值热力图、矢量箭头图、场线图，以及沿极轴方向的场强衰减曲线。C 程序则把计算结果导出为 CSV 文件，可以导入 Origin 绘制二维或三维分布图。

## 4. 图像结果讲解

从电偶极子的场强分布图可以看出，场强在偶极子附近最大，随着距离增大迅速衰减。热力图中颜色越亮的位置表示场强越强，颜色变暗则表示场强减小。

在矢量图中，箭头方向表示电场方向。对于沿 z 轴放置的电偶极子，场线从正电荷一侧出发，回到负电荷一侧，整体呈现出典型的偶极子分布形态。

磁偶极子的磁场线分布也具有类似的偶极子结构。它可以类比为一个很小的电流环或者条形磁体在远处产生的磁场。磁场在极轴附近较强，在远离源点后也会快速衰减。

程序中还绘制了沿极轴方向的 log-log 衰减曲线。理论上，静态偶极子的近场强度大致按照：

```text
1 / r^3
```

的规律衰减。数值结果与解析参考曲线基本一致，说明程序的计算结果是合理的。

额外加入的内容，比如三维切片图、CSV 数据导出、场强衰减验证和 C 语言版本，主要是为了让结果更容易检查、更容易复现，也更符合工程计算的表达方式。

从结果可以看出，电偶极子和磁偶极子的场都具有明显的轴对称特征，并且在远离源点时按照近似 `1/r^3` 的规律快速衰减。
